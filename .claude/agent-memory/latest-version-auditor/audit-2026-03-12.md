## Key Findings

- All remaining deps are actively imported in source code
- js-yaml usage is correct v4 API (load/dump, not deprecated safeLoad/safeDump)
- matrix-bot-sdk usage matches current 0.8.0 API patterns
- TypeBox 1.0 has breaking changes but Pi SDK constrains to ^0.34.41 — cannot upgrade
- hosted-git-info has no bundled types, @types/hosted-git-info is needed
- Biome used via npx (schema 2.4.6), not declared as dependency (intentional)
- Pi SDK packages correctly kept as peerDependencies
- Production dep tree is ~297 packages, nearly all from matrix-bot-sdk (express, crypto, etc.)
